Mini-Hackathon: Waste Management and Recycling in Indian Cities

Overview

Welcome to the Mini-Hackathon, organized by PWSkills, a leader in empowering aspiring data scientists with practical, hands-on learning experiences. This bi-monthly Hackathon is open to all enthusiasts, from beginners to seasoned practitioners, fostering innovation and technical expertise in data-driven solutions. Themed around Waste Management and Recycling in Indian Cities, this edition invites participants to tackle a critical real-world problem in urban sustainability using machine learning. By predicting recycling rates or exploring waste management insights, participants will contribute to efficient waste management practices and sustainable urban development while honing their data science skills.

Problem Statement

The objective is to develop a machine learning model that predicts the Recycling Rate (%) for a given set of city-specific waste management attributes, such as waste type, population density, and municipal efficiency. This is a regression problem aimed at minimizing prediction error (e.g., Root Mean Squared Error, RMSE). Using the Waste Management and Recycling in India dataset, participants will analyze features like waste generation, cost of management, and landfill capacity to derive actionable solutions for sustainable urban planning.

Dataset Overview

The dataset is a comprehensive collection of attributes related to waste generation, recycling, and disposal practices across Indian cities from 2019–2023. It includes various waste types (e.g., Plastic, Organic, E-Waste) and is sourced from simulated data inspired by municipal corporations and environmental reports.

Column Name Type Description
City/District Categorical Name of the Indian city or district (e.g., Mumbai, Delhi).
Waste Type Categorical Type of waste generated (e.g., Plastic, Organic, E-Waste, Construction, Hazardous).
Waste Generated (Tons/Day) Float Amount of waste generated in tons per day.
Recycling Rate (%) Float Percentage of waste that is recycled (primary target for regression).
Population Density (People/km²) Float Number of people per square kilometer in the city.
Municipal Efficiency Score (1-10) Integer Score indicating municipal waste management effectiveness.
Disposal Method Categorical Method used for waste disposal (e.g., Landfill, Recycling, Incineration, Composting).
Cost of Waste Management (₹/Ton) Float Cost of managing one ton of waste in Indian Rupees.
Awareness Campaigns Count Integer Number of waste management awareness campaigns organized in the year.
Landfill Name Categorical Name of the landfill site used by the city.
Landfill Location (Lat, Long) Float, Float Geographical coordinates of the landfill.
Landfill Capacity (Tons) Float Total waste capacity of the landfill in tons.
Year Integer Year of the data entry (2019–2023).

Task: Predict the “Recycling Rate (%)” using the provided features (regression). Challenges include handling categorical variables, addressing class imbalances, and ensuring robust feature engineering.

Tech Stack and Tools

Participants are encouraged to use the following tools and libraries, though flexibility is allowed for creative solutions:

  • Programming Language: Python (≥3.9)
  • ML Libraries: Scikit-learn, XGBoost/LightGBM/CatBoost, Joblib
  • Web Development: Flask, Streamlit (optional), Gunicorn
  • Data Visualization: Matplotlib, Seaborn, Plotly, Folium (optional)
  • Platforms: Google Colab, AWS, Render, Vercel, Hugging Face Spaces
  • Other Tools: Pandas, NumPy, GeoPandas (optional)
